# LockUSB
LockUSB is a Windows GUI tool that lets you enable or disable write protection on USB/storage drives using diskpart. Built with Python and customtkinter, it requires admin rights to run and offers a simple interface for drive management.

This Python script creates a LockUSB GUI application for Windows that allows users to enable or disable write protection (read-only mode) on USB drives or other storage devices using diskpart. The app uses customtkinter for a modern dark-themed interface and requires administrator privileges to function.

Key Features:

Displays available drives with protection status

One-click write protection enable/disable

Filters out network, CD-ROM, and floppy drives

Includes an About section with app details

Automatically requests admin rights if not run as administrator

Requirements: Windows OS, Python with customtkinter, and admin rights for disk operations.
